# PDF Generation

## Purpose

Render generated papers into downloadable PDF files.

## Technology

QuestPDF

## Input

Generated Paper Model

## Output

PDF Document

## Layout

Header:

* School Name
* Subject
* Date
* Total Marks

Body:

* Section A
* Section B
* Section C

Footer:

* Page Number
* Generated Timestamp

## Requirements

* Printable
* Consistent Layout
* Professional Formatting
* Fast Rendering
